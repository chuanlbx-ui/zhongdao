# ADR (Architecture Decision Records)

This folder stores project-level architecture and process decisions.

## Naming

Use `NNNN-short-title.md` (e.g. `0001-stability-governance.md`).

## Template

```md
# ADR NNNN: Title

## Status
Accepted | Proposed | Superseded

## Context
Problem, constraints, and background.

## Decision
What is chosen.

## Consequences
Positive, negative, and tradeoffs.

## Rollout Plan
How the decision is implemented.
```
